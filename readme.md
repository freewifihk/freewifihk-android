# FreeWifiHK

Discover free Wi-Fi hotspots in Hong Kong.

## Usage

1. Convert the [FreeWifiHK-DB][db] to JSON in `assets/hotspots.json`
2. Build using Eclipse

  [db]: https://github.com/freewifihk/freewifihk-db
